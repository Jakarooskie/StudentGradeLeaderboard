# Student Leaderboard

**Author:** Jacob Vescio  
**Date:** 2026-03-28  

## Description
This is a C program that creates a leaderboard for students based on their grades.  
Users can input up to 100 students and choose to display the leaderboard from Highest to Lowest or Lowest to Highest.  

## Features
- Input student names (supports spaces) and grades.
- Sort leaderboard in ascending or descending order.
- Displays final leaderboard in the terminal.
